Create the custom post type.

== Description ==

This plugin to easy create custom post types.

カスタム投稿タイプを作成するプラグインです。  
必要な項目を選択するだけで簡単に作成することができます。

配布している WordPress Plugin に関する情報を配信する Facebook ページを作成しました。  
ぜひ『いいね！』を押してください。

https://www.facebook.com/pages/WordPress-Plugins/563277537040790

== Installation ==

1. Upload `custom-post-type-manager` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
